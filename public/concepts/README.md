# Concept photos

Place concept images in `public/concepts/images/`.

Files in `public/` are served from the site root, so an image stored at:

```txt
public/concepts/images/ritual.webp
```

can be used in the app as:

```txt
/concepts/images/ritual.webp
```

Recommended format: optimized `.webp` images with descriptive lowercase slugs, such as `ritual.webp`, `craft.webp`, or `memory-map.webp`.
